## Evidence

- Boundary is respected: the factory remains pure TypeScript with no Electron import, no `ipcMain.handle()` registration, no `BrowserWindow`, no renderer UI, no EventSource/SSE client, no token/port persistence, no `runtime.lock` schema change, and no production dependency change.

- `blocked` / `timed_out` results fail closed through `RuntimeStartupUnavailableError`, and `stop()` returns `false` unless a ready startup result exists.
- `reused_existing` sidecar ownership remains safe because the factory delegates to the controller result where `stop()` returns `false`.
- Unknown fetch IPC payload validation covers object shape, path type, init shape, scalar field types, header value types, method allowlist, relative path rules, reserved headers, and CR/LF-safe headers.
