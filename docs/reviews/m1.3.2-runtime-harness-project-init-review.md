## Independent Review

Initial independent subagent verdict: `FAIL`.

Findings:

- `[P1]` Initial CW commit could include pre-staged user files.
- `[P1]` `skills.config.json`, `mcp.config.json`, and `adapters.config.json` were objects instead of list-of-objects manifests.
- `[P2]` malformed JSON on `POST /cw/v1/projects` escaped the endpoint and returned a plain 500 instead of `ErrorEnvelope`.

Repair applied:

- Changed the first CW commit to `git commit --only -- <CW generated paths>` and added a pre-staged user file regression test.
- Changed the three config manifests to initialize as JSON arrays and added assertions.
- Wrapped JSON parsing in `POST /projects`, required request `schema_version`, and added malformed JSON / missing schema tests.

Follow-up independent subagent verdict: `PASS`.
