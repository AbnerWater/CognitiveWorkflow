## Confirmed Properties

- The DOM session composes only an injected W1.5.75 shell adapter and the W1.5.77 DOM keyboard adapter.
- The module does not read or write global `window` or `document`.
- The module does not import React, Electron, IPC, storage APIs, EventSource, fetch, or any new dependency.
- `bindKeyboardTarget()` fails closed once the session or adapter is disposed.
- Rebinding replaces the prior keyboard target listener before registering the next one.
- `unbindKeyboardTarget()` clears the stored unsubscribe before invoking it and remains idempotent.
- `dispose()` unbinds the keyboard target, disposes the injected shell adapter, and remains idempotent.
- The session does not own or dispose the upstream workbench host/session behind the injected adapter.
- After dispose, subscriptions and keyboard binding fail closed; adapter actions continue to use the adapter's disposed fail-closed behavior.
