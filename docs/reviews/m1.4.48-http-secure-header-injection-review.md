## Boundary Check

- This slice implements explicit secure-store backed header injection for MCP HTTP discovery/invocation clients.
- It does not implement OAuth refresh/token exchange, stdio MCP `env` injection, AdapterRegistry/API wiring, public or third-party provider validation, real provider upstream `cancel()`, Claude Agent SDK real-provider HITL, or end-to-end runtime API MCP invocation.

## Residual Risk
