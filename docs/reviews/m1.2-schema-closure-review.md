## Error Code Scope

W1.2.8 covers 35 schema-layer spec custom error codes:

- WG: 13
- NC: 6
- CP: 2
- EP: 3
- ER: 4
- RP: 4
- SE: 3

The guard intentionally excludes:

- L1 raw JSON parse errors handled before Pydantic model validation.
- Compiler/reachability and L4 external registry errors.
- Runtime/apply/persist/SSE state errors owned by runtime harness, API, persistence, or stream services.
- Native Pydantic literal/enum/range validation where no schema spec custom error is locked.
