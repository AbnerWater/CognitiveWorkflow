## Findings

- Fix: `cloneRuntimeStreamEvent()` now uses `structuredClone(event.data)` for payload-level isolation.
- Regression coverage: the capped-buffer test mutates both the original emitted payload and a returned snapshot payload, then asserts the retained store event remains unchanged.
