## W1.2.7 产出

- `scripts/codegen/generate-ts-schemas.mjs` 已从 schema-name stub 升级为真实 `json-schema-to-typescript` 编译器。
- `scripts/codegen/format-json-schemas.mjs` 显式格式化 Pydantic JSON Schema 产物，避免 TS generator 隐式修改输入。
- `packages/schemas-ts/src/generated/*.ts` 现在导出对应 Pydantic 模型的 TypeScript 类型，而不是 `*_SCHEMA_NAME` 占位常量。
- `packages/schemas-ts/src/generated/index.ts` 作为 generated barrel，使用 `.js` ESM specifier；`packages/schemas-ts/src/index.ts` 通过 `export type *` 对外暴露 90 个生成类型。
- `json-schema-to-typescript` 作为 workspace 根 devDependency，仅用于 codegen。
